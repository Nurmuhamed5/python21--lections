ls показывает все файлы и папки внутри текущей директор
ls test показывает все файлы и папки внутри папки test
ls -a показывает также скрытые файлы и папки
ls -l показывает более подробную информацию

cd - переходит в домашнюю папку
cd переходит в домашнюю папку
cd test - переходит в папку test
cd. - переходит в текущую папку (остается  вней же)
cd.. - переходит в предыдущую папку
cd test/test2 переходит в папку 

mkdir test - создает папку test
mkdir test/test2 - создает папку test2 внутри папки test (если папка tets существует)
mkdir test1 test2 - создает папки test1 и test2

touch test.text - создает файлы test/text
touch test/txt.

rm test - удаляет файлы test (если это не папка)
rm -rf test - удаляет файл или папку со всем содержимымм(удаляет безвозратно)
rm - rf / никогда не писать

pwd - показывает где мы сейчас находимся (print working directory)

cat test.txt - поазывает содержимое файла

nano test.txt - открывает test.txt в редакторе когда nano (в терминале)

code test.txt - открывает test.txt  в редакторе когда vscode
